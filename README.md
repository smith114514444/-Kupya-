## ✨ 特色亮点

- 🧸 **自主移动**  
  库皮亚会在桌面上自由散步、闲逛，偶尔发呆。

- 📺 **内置电视系统**  
  没事的时候它会自己放置小电视看视频！你可以把自己喜欢的**ogv**格式的视频放入 `video` 文件夹，库皮亚会随机播放。  
  *鼠在电视机上滑动滚轮调节音量*

- 🎨 **可自定义贴纸**  
  把贴纸图片放入 `sticker` 文件夹，在装扮菜单中装饰库皮亚。  
  *鼠标滚轮 = 调整贴纸大小*  
  *鼠标侧键 = 旋转贴纸* （自定义按键即将补全）

- 🎭 **好感度剧情（开发中）**  
  当前版本（0.1.3.1）为好感度对话封锁版本。完整对话版暂不外发，因为剧情较敏感，需内部打磨。  
  欢迎加入内群品尝完整版，并提供角色设定反馈！  

- 🛠️ **完全开源 · Godot 架构**  
  基于 Godot 引擎开发，代码结构混乱，我就不信有人能二次创作或学习桌面宠物实现逻辑(千万别学，小心变成屎山大王)。

## 🖥️ 运行要求

- **操作系统**：支持 Windows（7 / 10 / 11）  
- **去安装.NET10SDK** —— 下载后直接双击 `Kupya.exe` 即可运行。

## 🎮 交互指南

| 操作对象 | 交互方式 | 效果 |
|---------|----------|------|
| 电视机窗口 | 鼠标滚轮 | 调节音量 |
| 贴纸 | 鼠标滚轮 | 放大/缩小 |
| 贴纸 | 鼠标侧键 | 旋转角度（自定义按键即将支持） |
| 库皮亚本体 | 拖动/点击 | 拖拽移动 / 待开发互动 |

## 📁 自定义内容

你可以轻松为库皮亚增加新内容：

- **视频**：把 `.ogv` 或其他 Godot 支持的视频文件放入 `video` 文件夹，电视机会自动识别并随机播放。
- **贴纸**：把图片文件（`.png`）放入 `sticker` 文件夹，即可在桌宠界面生成可调节的贴纸。
